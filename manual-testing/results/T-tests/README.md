# Archived T-Test Results

This folder stores historical `T-tests` run outputs that should not live inside the source control pack.

Keep the source pack under [manual-testing/T-tests](/C:/Users/Calvin/Software%20Projects/LocalAgent/manual-testing/T-tests) limited to:
- task fixtures
- pack documentation
- `results/RESULTS_TEMPLATE_T.md`

Put archived run outputs here instead, for example:
- `RESULTS_T_RUN_*.md`

This keeps the pack hygiene check aligned with the intended control-pack contract.
